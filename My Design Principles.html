Aidan's Design Principles

Cultural Concepts

  • Avoid using words that translate to a different meaning in other languages
  • Research meanings of certain colors - colors having different meanings in other cultures
  • Modify a design to cater to direction in which the language is read

Learning and Memory Concepts

  • Use metaphors to help user connect your design to real life concepts
  • Create a design similar to other popular designs because users spend most their time with other designs - Jakob's Law
  • Use constraints to guide new users
  • Allow users to remember no more than 7 things - this is the magic memory number
  • Allow users to label information abundantly (color, name, location)

Perception Concepts

  • Use icons
  • Play unique audio to inform user more
  • Implement smart color techniques
  • Make important information stand out by using color, underlining, animation, and spacing

Functionality Concepts

  • Make what is clickable obvious by using affordances
  • Use signifiers to guide the users
  • Use immediate feedback to show user what they've done
  • Map your controls to actions well
  • Adapt your design to the users, don't make users adapt to design
  • Guide users to functions they need most

Cognitive Concepts

  • Don't make users think
  • Break complex task into smaller tasks
  • Display information in bite sized chunks - people process this information better
  • Let user learn by doing - people learn best from examples
  • Avoid overwhelming users by highlighting recommended options
  • Provide help information pages

Accessibility Concepts

  • Create a design that is easy to understand, especially by an inexperienced user
  • Design to be accessible regardless of disability
  • Consider font sizes, test amount, colors
  • Use multiples cues
  • Give clear instructions to the user
  • Allow for text resizing that won't affect formatting
  • Reduce speech-based menus

Emotional Concepts

  • Understand users' wants, needs, and preferences
  • Design for the targeted audience (age, interests, knowledgeâ€¦)
  • Create a comfortable, fun design
  • Focus on a good look and feel because users use this as the first indicator of trust
  • Use stories and surprises to engage the user

Building for Scanning

  • Create visual hierarchies
  • Break pages into areas
  • Eliminate distractions
  • Omit needless words

Designing for Smartphones

  • Minimize clutter
  • Avoid long dropdowns
  • Provide easy access to common tasks
  • Allow for zooming or resizing
  • Keep buttons in comfortable positions for fingers - make one handed navigation easy
